# Dependency rationale

- `@opentui/core` provides the terminal renderer and native terminal lifecycle.
- `@opentui/react` provides the React reconciler, intrinsic terminal components, and keyboard hooks.
- `react` provides composable UI state and component boundaries.
- `typescript` 7 provides strict compile-time contracts and fast checks.
- `oxlint` provides the focused lint gate.
- `prettier` provides deterministic formatting.
- `@types/bun`, `@types/node`, and `@types/react` provide type declarations for the Bun runtime, local sockets/files, and React components.
